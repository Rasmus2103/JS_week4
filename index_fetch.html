Reflections on Exercise-1

What is a Promise?

A promise is a Javascript object that represent a completion of an asynchronous operation and its resulting value.

What does the fetch method return

The fetch method returns a Promise that contains the completed response to the request. However it does not contain the actual body of the of the response. We can use methods like .json() to get body of that specific response we wish to pick up. In this Exercise we are using it the get the body of the users from of specific api.

Why do we need to have two .then callbacks with our fetch code?

We have 2 .then callback because there are necessarys steps to complete our fetch request

The first step is that we need the actual network request itself, needs to be resolved into a Response object.

The second step is processing the body of the response, and we can do that ny reading it as JSON.

What can and should you do in the first?

The thing we can and should do first is to check if request was successfull. We can do by either using the ok property, or as its more commonly used, we can read with JSON and the content from api will be showed as JSON objects.

What can and should you do in the last?

The thing we can and should do last, is to handle the data the are send to us from the api. If the request was successfull, we can begin with how we wish to handle our parsed data, in this case JSON data. In this Exercise, we have in our second .then callback parsed the data, and then convert it into HTML objects.
