    Reflections on exercise-3

    This Exercise is about using the map function to create a list from an arraylist in Javascript, and then convert over to pure html, and also how we control the content of the list. Also on how keep the site from reloading everytime we try to submit something.

    We have specifically used the .preventDefault() in this exercise to prevent to site from reloading everytime we either submit a name in input-tag, try to delete a name from first place or try to delete a name from the last place. In the form-tag of our HTML-file, our buttons does not have any "type" specifically set to any of those button-tag, so everytime we try to submit with the ' buttons, the site would keep reloading, and not take any of the data we are giving it.
